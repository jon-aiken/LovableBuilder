# Design Toolbox

A curated set of tools, resources, and communities for Lovable builders. Use these to source components, find inspiration, generate assets, and level up your prompting.

---

## Component Libraries

Copy code or prompts → Get 5 results. Browse these libraries, find what you want, copy the code or prompt, and paste it directly into Lovable.

### [21st.dev](https://21st.dev)
Production-ready React component libraries with Tailwind. Browse components, copy code, paste into Lovable.
- Best for: Navigation, hero sections, cards, forms, dashboards

### [ReactBits](https://reactbits.dev)
Animated React components and interactions. Pre-built with Framer Motion and Tailwind.
- Best for: Animated hero sections, micro-interactions, scroll effects

### [Hover.dev](https://hover.dev)
Hover effects and animations ready for Tailwind + React.
- Best for: Button hover effects, card animations, interactive elements

## Design Inspiration

Find references → Combine with prompts. Screenshot designs you like, then attach them to your Lovable prompts as visual references.

### [Dribbble](https://dribbble.com)
High quality design inspiration from designers worldwide.
- Best for: UI patterns, colour palettes, overall aesthetic direction

### [Mobbin](https://mobbin.com)
Real app UI and UX patterns — screenshots from live production apps.
- Best for: Flow references, actual product UI patterns, feature design

### [LandingFolio](https://landingfolio.com)
Curated landing page inspiration and component previews.
- Best for: Landing page layouts, hero sections, pricing tables

## AI Generation

**Generate → Animate → Import.** Create images or videos with AI tools, then upload them to Lovable for dynamic hero sections and visual assets.

### [Aura.build](https://aura.build)
AI design tool with templates. Generate UI mockups and visual designs.
- Use: Generate design mockups → upload to Lovable as reference images

### [Google Flow Veo 3](https://labs.google/flow)
AI video generation for hero sections.
- Use: Generate video backgrounds → import to Lovable for hero animations

### [Midjourney](https://midjourney.com)
AI image generation & visual concepts.
- Use: Generate brand imagery, illustrations, hero visuals → import to Lovable

**Workflow:** Create an image or video in these tools → attach to your Lovable prompt → describe the element you want. Lovable will match the visual reference precisely.

## Prompting with Design References — Tips

1. **Screenshot → Attach → Prompt**: Find a design you like (Dribbble, Mobbin, or a live site), screenshot it, attach it to your Lovable prompt with: "Match the layout style of this design. Apply our brand colours and typography from docs/design.md."

2. **Component → Copy → Paste**: Find a component on 21st.dev or ReactBits, copy the code, and paste it into Lovable Code Editor. Tell Lovable: "Integrate this component into [page]. Adapt it to use our design tokens."

3. **Generate → Import**: Use Aura.build or Midjourney to generate visuals for your hero section. Upload to Supabase Storage, then reference the URL in your Lovable prompt.

4. **Buzzwords as prompts**: Felix Haas's key insight — design vocabulary works better than descriptions. "Glassmorphism card with subtle backdrop-blur" beats "a card that looks kind of glass-like." Build a vocabulary of terms that consistently produce results you like.
