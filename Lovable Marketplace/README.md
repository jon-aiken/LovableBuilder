## Add This Marketplace

In Claude Code, run:

```
/plugin marketplace add jon-aiken/LovableBuilder
```

---

## Available Plugins

### `lovable-prd-builder` — The Complete Lovable.dev Build System

The full 9-step workflow from idea to published app, built around Lovable's Documentation-First approach.

**Install:**
```
/plugin install lovable-prd-builder@agent-builder-academy
```

**Two skills included:**

| Skill | Steps | What it does |
|-------|-------|-------------|
| `lovable-prd-builder` | 1–3 | Discovery interview → 4 PRD documents (Masterplan, Implementation Plan, Design Guidelines, App Flow) |
| `lovable-build-guide` | 4–9 | Knowledge Base setup, Magic Prompt build loop, maintenance, precision editing, deployment |

**Two commands:**
- `/lovable-prd-builder:build-prd` — Start the PRD builder
- `/lovable-prd-builder:lovable-guide` — Start the build execution guide
